# AquaPercent AI - Predictive Water Intelligence System

Enterprise-grade AI platform for water infrastructure monitoring, leak prediction, and intelligent decision support.

## Features

- **Cinematic Intro Experience** - Professional landing page with animated metrics
- **Global Command Center** - Real-time infrastructure monitoring with live charts
- **Leak Prediction Engine** - AI-powered predictive maintenance with risk scoring
- **Infrastructure Map** - Interactive pipeline visualization with status indicators
- **Decision Intelligence** - Automated recommendations with urgency tracking
- **Citizen Portal** - Public-facing water information interface
- **Farmer Mode** - Agricultural water management and irrigation guidance
- **AI Insights Lab** - Machine learning model analytics and performance
- **System Analytics** - Impact metrics and ROI tracking
- **Dual Theme System** - Dark Intelligence & Eco Sustainability modes

## Tech Stack

- Next.js 14 (React 18)
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Recharts (data visualization)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```
